Создание консольного приложения для рисования с использованием объектно-ориентированного программирования (ООП) на Python — это интересная задача. Ниже приведены функциональные требования для такого приложения, включая команды undo, redo, save, load, добавление фигур (треугольник, прямоугольник, круг), рисование и помощь.

▎Функциональные требования

▎1. Общие требования

• Приложение должно быть реализовано на Python с использованием ООП.

• Интерфейс пользователя должен быть консольным.

• Все команды должны быть доступны через командную строку.

▎2. Команды

▎2.1. add triangle

• Описание: Добавляет треугольник в текущий холст.

• Параметры:

  • Координаты вершин треугольника (x1, y1), (x2, y2), (x3, y3).

• Результат: Треугольник добавляется в список фигур на холсте.

▎2.2. add rectangle

• Описание: Добавляет прямоугольник в текущий холст.

• Параметры:

  • Координаты верхнего левого угла (x, y), ширина и высота.

• Результат: Прямоугольник добавляется в список фигур на холсте.

▎2.3. add circle

• Описание: Добавляет круг в текущий холст.

• Параметры:

  • Координаты центра круга (x, y) и радиус.

• Результат: Круг добавляется в список фигур на холсте.

▎2.4. draw

• Описание: Рисует все фигуры на холсте.

• Результат: Консоль выводит текстовое представление всех фигур, добавленных на холст.

▎2.5. undo

• Описание: Отменяет последнее действие (добавление фигуры).

• Результат: Последняя добавленная фигура удаляется из списка фигур.

▎2.6. redo

• Описание: Повторяет последнее отмененное действие.

• Результат: Последняя отмененная фигура снова добавляется на холст.

▎2.7. save

• Описание: Сохраняет текущее состояние холста в файл.

• Параметры:

  • Имя файла для сохранения.

• Результат: Текущие фигуры сохраняются в указанном файле в формате JSON или текстовом формате.

▎2.8. load

• Описание: Загружает состояние холста из файла.

• Параметры:

  • Имя файла для загрузки.

• Результат: Фигуры из указанного файла загружаются на холст.

▎2.9. help

• Описание: Выводит справочную информацию о доступных командах и их использовании.

• Результат: Консоль выводит список доступных команд и их описание.
